# Teammate AI Quickstart (Human Guide)

Purpose: This file is for human teammates to copy prompts quickly. It is not a project policy file.

---

## Session Start Prompt (Copy and Edit)

```text
You are my coding copilot for Team 3.
My role: <Role X>, owner: <Your Name>.
Task for this session: <deliverable>.

Before coding, read in order:
1) docs/README.md
2) docs/Decision-Log.md
3) docs/Team-Workflow-and-Roles.md
4) docs/AI-Assistant-Handoff.md

Rules:
- Do not modify frozen decisions.
- Keep changes small and modular.
- Work only within my role scope unless I explicitly ask cross-role edits.

After edits, report:
A) code changes
B) Decision Log entries needed (if behavior/principle changed)
C) suggested commit message

Start with a 5-line plan.
```

---

## 30-Second Checklist

1. Replace role and owner.
2. Replace deliverable with tonight's target.
3. Confirm AI read order exactly.
4. Ask for Decision Log vs commit split before finishing.
